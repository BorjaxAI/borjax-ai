Changelog

All notable changes to BorjaxAI. We follow semantic versioning.

v0.1.0
May 3, 2026 — Initial Release 🎉
New Public Beta

This is the first public release of BorjaxAI — a full AI platform with chat, tasks, and agents. The platform is stable and ready for production use.

✨ New Features

  • AI Chat with Streaming — Real-time streaming chat powered by Claude 3.5 Sonnet via AWS Bedrock. Full Server-Sent Events (SSE) support with conversation history and context memory.
  • Background Tasks — Async task queue via Celery + Redis. Four built-in task types: research, write, analyze, scrape. Custom agent tasks supported.
  • Agent Builder — No-code interface and API for creating custom AI agents with roles, goals, backstories, and tool access. Powered by CrewAI with pre-built crew templates.
  • Guest Mode — Try BorjaxAI without an account. 5,000 free tokens per guest session, valid for 24 hours.
  • Freemium Billing — Four plans: Free (10K tokens), Starter ($9/mo, 100K), Pro ($29/mo, 500K), Agency ($99/mo, 2M). Powered by Stripe.
  • REST API — Full API with JWT authentication, SSE streaming, and consistent JSON responses. Base URL: https://api.borjaxai.com/v1
  • Web Application — Full-featured React + TypeScript frontend deployed on CloudFront at d18813a3in0ap7.cloudfront.net.
  • Documentation Site — Comprehensive docs with 3-column layout, live search, copy buttons, callouts, and TOC navigation.

🏗️ Infrastructure

  • API Gateway: Hono (TypeScript) on AWS Lambda + API Gateway. Handles auth, routing, and Stripe billing.
  • AI Backend: FastAPI (Python) on AWS ECS Fargate. Handles LLM calls, CrewAI orchestration, and PostgreSQL reads/writes.
  • Task Queue: Celery workers on ECS with Redis (ElastiCache) as broker and result backend.
  • Database: PostgreSQL 16 on RDS with pgvector extension for future semantic search.
  • LLM: AWS Bedrock — Claude 3.5 Sonnet (anthropic.claude-3-5-sonnet-20241022-v2:0). No OpenAI dependency.
  • Frontend: React 18 + TypeScript + Vite + Tailwind CSS, deployed to S3 + CloudFront.

🔒 Security

  • JWT-based authentication with HS256 signing and configurable expiry
  • Password hashing via bcrypt
  • Rate limiting on all auth endpoints (5 attempts per minute per IP)
  • CORS configured with strict origin allow-list
  • All traffic over HTTPS/TLS 1.3

⚠️ Known Limitations

  • Conversation history retention on Free plan is limited to 30 days
  • Guest sessions cannot be upgraded to a registered account mid-session (create a new account instead)
  • The browser tool for agents is in beta and may be less reliable on JS-heavy pages
  • No webhooks yet — poll for task status (coming in v0.2.0)
  • No file upload support yet (coming in v0.2.0)

🔮 Coming in v0.2.0

  • Webhooks: receive a POST when a task completes
  • File uploads: attach PDFs, CSVs, and images to tasks
  • Task scheduling: run tasks on a cron schedule
  • Agent templates marketplace
  • Usage analytics dashboard
  • Team collaboration features on Pro plan
