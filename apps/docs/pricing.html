What Counts as a Token?

BorjaxAI uses Claude's native token counting. Roughly speaking, 1 token ≈ 4 characters of English text. Tokens are consumed on both the input (what you send) and the output (what Claude returns).

  • A short chat message ("Summarize this article") uses ~8 input tokens
  • A 500-word article Claude writes uses ~700 output tokens
  • A research task (input + multi-agent reasoning + output) uses 2,000–6,000 tokens
  • Code files and structured data are tokenized the same as text

Frequently Asked Questions

Can I upgrade or downgrade at any time?

Yes. Upgrades take effect immediately and you're billed a prorated amount for the remainder of the billing cycle. Downgrades take effect at the start of your next billing cycle.

What happens when I hit my token limit?

The API will return a 402 Payment Required error with code TOKEN_LIMIT_REACHED. Chat, tasks, and agent runs will all fail until you upgrade or your monthly limit resets. You can check your current usage at any time via GET /v1/auth/me.

Do unused tokens roll over?

No. Token allowances reset on the first day of each billing cycle. Unused tokens do not carry over to the next month.

What about guest tokens?

Guest sessions (no account) receive 5,000 one-time tokens. These are not monthly — they're a one-time allowance per browser session. Create a free account to get 10,000 monthly tokens.

Can I purchase additional tokens?

Additional token top-ups are not currently available. To get more tokens, upgrade to the next plan. If you need a custom volume arrangement (enterprise usage), contact us about the Agency plan.
