API: Chat

Stream Claude responses in real time using Server-Sent Events (SSE). Manage conversation threads and retrieve message history.

Send Message

POST /v1/chat/send Auth Required SSE Stream

Send a message and receive a streaming SSE response. Set conversation_id to null to start a new conversation, or pass an existing ID to continue one.

Request Body

json
{
  "message": "What are the top 3 benefits of TypeScript?",
  "conversation_id": null
}
FieldTypeRequiredDescription
messagestringYesThe user's message to Claude
conversation_idstring | nullYesExisting conversation UUID to continue, or null to start new

SSE Response Format

The response is a stream of text/event-stream events. Each line starts with data: followed by a JSON object.

text
data: {"type":"token","content":"Type"}
data: {"type":"token","content":"Script"}
data: {"type":"token","content":" provides"}
data: {"type":"token","content":" static"}
...
data: {"type":"done","conversation_id":"abc123-def456","tokens_used":248}

SSE Event Types

TypeFieldsDescription
tokencontent: stringA partial text token from Claude. Append to your buffer to reconstruct the full response.
doneconversation_id, tokens_usedStream complete. Contains the conversation ID for follow-up messages and the total tokens consumed.
errormessage, codeAn error occurred. See error codes below.

Full cURL Example

bash
curl -X POST https://api.borjaxai.com/v1/chat/send \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: text/event-stream" \
  -N \
  -d '{
    "message": "Explain async/await in JavaScript.",
    "conversation_id": null
  }'

Full JavaScript Streaming Example

javascript
async function chat(message, conversationId = null) {
  const response = await fetch('https://api.borjaxai.com/v1/chat/send', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${ACCESS_TOKEN}`,
      'Content-Type': 'application/json',
      'Accept': 'text/event-stream',
    },
    body: JSON.stringify({ message, conversation_id: conversationId }),
  });

  if (!response.ok) {
    const err = await response.json();
    throw new Error(err.detail || 'Chat request failed');
  }

  const reader = response.body.getReader();
  const decoder = new TextDecoder();
  let buffer = '';
  let finalConvId = null;

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;

    const chunk = decoder.decode(value, { stream: true });
    for (const line of chunk.split('\n')) {
      if (!line.startsWith('data: ')) continue;
      const raw = line.slice(6).trim();
      if (!raw) continue;
      const event = JSON.parse(raw);

      if (event.type === 'token') {
        buffer += event.content;
        process.stdout.write(event.content); // or append to DOM
      } else if (event.type === 'done') {
        finalConvId = event.conversation_id;
        console.log(`\nTokens used: ${event.tokens_used}`);
      } else if (event.type === 'error') {
        throw new Error(event.message);
      }
    }
  }

  return { text: buffer, conversationId: finalConvId };
}

// Usage
const { text, conversationId } = await chat('Hello! What is BorjaxAI?');
// Continue the same conversation
await chat('Tell me more about the Agent Builder.', conversationId);

Full Python SSE Example

python
import requests
import json

def chat(message: str, conversation_id: str | None = None, access_token: str = ""):
    response = requests.post(
        "https://api.borjaxai.com/v1/chat/send",
        headers={
            "Authorization": f"Bearer {access_token}",
            "Content-Type": "application/json",
            "Accept": "text/event-stream",
        },
        json={"message": message, "conversation_id": conversation_id},
        stream=True,
    )
    response.raise_for_status()

    buffer = ""
    final_conv_id = None

    for line in response.iter_lines():
        if not line:
            continue
        decoded = line.decode("utf-8")
        if not decoded.startswith("data: "):
            continue
        event = json.loads(decoded[6:])

        if event["type"] == "token":
            buffer += event["content"]
            print(event["content"], end="", flush=True)
        elif event["type"] == "done":
            final_conv_id = event["conversation_id"]
            print(f"\nTokens used: {event['tokens_used']}")
        elif event["type"] == "error":
            raise RuntimeError(event["message"])

    return buffer, final_conv_id

# Usage
text, conv_id = chat("Summarize async programming in Python.", access_token="YOUR_TOKEN")
text2, _ = chat("Give an example.", conversation_id=conv_id, access_token="YOUR_TOKEN")

List Conversations

GET /v1/chat/conversations Auth Required

Returns a paginated list of all conversations for the authenticated user, ordered by most recent.

Query Parameters

ParamTypeDefaultDescription
pageinteger1Page number
limitinteger20Results per page (max 50)

Response 200

json
[
  {
    "id": "abc123-def456",
    "title": "Async programming in Python",
    "message_count": 6,
    "tokens_used": 1842,
    "created_at": "2026-05-03T12:00:00Z",
    "updated_at": "2026-05-03T12:05:33Z"
  }
]

Error Handling

If you hit the token limit mid-stream, the SSE stream will emit an error event before closing:

text
data: {"type":"error","code":"TOKEN_LIMIT_REACHED","message":"Monthly token limit exceeded. Upgrade your plan to continue."}

Handle it in your stream reader and prompt the user to upgrade.
