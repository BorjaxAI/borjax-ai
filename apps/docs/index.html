BorjaxAI Documentation

Welcome to BorjaxAI — a full AI platform that gives you streaming chat, background tasks, and custom AI agents, all powered by Claude 3.5 Sonnet on AWS Bedrock.

What is BorjaxAI?

BorjaxAI is a developer-first SaaS platform designed to make AI capabilities accessible without the infrastructure overhead. Whether you're building a product, automating workflows, or experimenting with large language models, BorjaxAI gives you three powerful primitives: a real-time chat interface, an asynchronous task queue, and a no-code agent builder — all accessible via a clean REST API.

Under the hood, BorjaxAI routes every inference request through AWS Bedrock using Claude 3.5 Sonnet. A FastAPI backend handles LLM calls and agent orchestration via CrewAI, while a Hono-based TypeScript gateway handles authentication, billing (Stripe), and routing. Background tasks are powered by Celery + Redis so your long-running AI jobs never block your users.

ℹ️
Note: BorjaxAI is in public beta. Core APIs are stable. Some advanced features are still being developed — see the Changelog for the latest.

Getting Started in 3 Steps

Try it free — no account needed
Visit the app and start chatting immediately in guest mode. You get 5,000 free tokens with no registration required. Your session is valid for 24 hours.
Explore every feature
Create a free account to unlock 10,000 monthly tokens, background tasks, and your first AI agent. Upgrade to Starter, Pro, or Agency when you need more.
Integrate via API
Use the REST API to embed BorjaxAI into your product. Authenticate with a JWT bearer token, call any endpoint, and stream responses with SSE.
