API: Agents

Create, manage, and run custom AI agents. Each agent is a reusable persona with a role, goal, backstory, and tool access. Running an agent creates a background task.

Create Agent

POST /v1/agents Auth Required

Create a new custom agent. The agent is saved to your account and can be run any number of times.

Request Body

json
{
  "name": "Competitive Intelligence Analyst",
  "role": "Senior Competitive Intelligence Analyst",
  "goal": "Analyze competitor products, pricing, and strategies to identify opportunities and threats",
  "backstory": "You are an experienced competitive intelligence professional with 10+ years at top-tier consulting firms. You specialize in SaaS markets and have developed frameworks for rapid competitive analysis. You always provide data-backed insights with clear business implications.",
  "tools": ["web_search", "summarize", "analyze"],
  "verbose": false
}
FieldTypeRequiredDescription
namestringYesFriendly display name for the agent
rolestringYesProfessional title (e.g., "Senior Data Analyst")
goalstringYesWhat the agent optimizes for — guides CrewAI task execution
backstorystringRecommendedRich background that shapes Claude's persona — key for quality
toolsstring[]NoTools the agent can use. See tools reference below.
verbosebooleanNoDefault false. Set true to include agent reasoning in the result.

Response 201 Created

json
{
  "id": "agent_8a3f2d1e",
  "name": "Competitive Intelligence Analyst",
  "role": "Senior Competitive Intelligence Analyst",
  "goal": "Analyze competitor products, pricing, and strategies...",
  "backstory": "You are an experienced competitive intelligence professional...",
  "tools": ["web_search", "summarize", "analyze"],
  "verbose": false,
  "created_at": "2026-05-03T12:00:00Z",
  "updated_at": "2026-05-03T12:00:00Z"
}

List Agents

GET /v1/agents Auth Required

Returns all agents belonging to the authenticated user.

bash
curl https://api.borjaxai.com/v1/agents \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

Response 200

json
[
  {
    "id": "agent_8a3f2d1e",
    "name": "Competitive Intelligence Analyst",
    "role": "Senior Competitive Intelligence Analyst",
    "tools": ["web_search", "summarize", "analyze"],
    "created_at": "2026-05-03T12:00:00Z"
  }
]

Update Agent

PUT /v1/agents/{id} Auth Required

Update any field on an existing agent. All fields are optional — only send the fields you want to change.

bash
curl -X PUT https://api.borjaxai.com/v1/agents/agent_8a3f2d1e \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "tools": ["web_search", "summarize", "analyze", "browser"],
    "verbose": true
  }'

Returns the full updated agent object with 200 OK.

Delete Agent

DELETE /v1/agents/{id} Auth Required

Permanently delete an agent. This does not delete tasks that were previously run with this agent.

bash
curl -X DELETE https://api.borjaxai.com/v1/agents/agent_8a3f2d1e \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

Returns 204 No Content on success.

Run Agent

POST /v1/agents/{id}/run Auth Required

Execute the agent with a given prompt. Creates a background task and returns the task_id immediately. Poll GET /v1/tasks/{task_id} for the result.

Request Body

json
{
  "prompt": "Analyze BorjaxAI's top 3 competitors: their pricing, feature gaps, and positioning."
}

Response 202 Accepted

json
{
  "task_id": "task_2b4f8e1a"
}

Full End-to-End Example

Create an agent, run it, and retrieve the result in one flow:

javascript
const BASE = 'https://api.borjaxai.com/v1';
const headers = {
  'Authorization': `Bearer ${ACCESS_TOKEN}`,
  'Content-Type': 'application/json',
};

// 1. Create the agent
const createRes = await fetch(`${BASE}/agents`, {
  method: 'POST',
  headers,
  body: JSON.stringify({
    name: 'Market Researcher',
    role: 'Senior Market Research Analyst',
    goal: 'Research markets and deliver actionable intelligence reports',
    backstory: 'You are a seasoned analyst with 12 years at top consulting firms. You synthesize complex information into clear, actionable insights with supporting data.',
    tools: ['web_search', 'summarize'],
  }),
});
const agent = await createRes.json();
console.log('Agent created:', agent.id);

// 2. Run the agent
const runRes = await fetch(`${BASE}/agents/${agent.id}/run`, {
  method: 'POST',
  headers,
  body: JSON.stringify({
    prompt: 'Research the AI developer tools market in 2026: size, key players, growth drivers, and emerging opportunities.',
  }),
});
const { task_id } = await runRes.json();
console.log('Task submitted:', task_id);

// 3. Poll for result
async function poll(taskId, delay = 5000) {
  await new Promise(r => setTimeout(r, delay));
  const res = await fetch(`${BASE}/tasks/${taskId}`, { headers });
  const task = await res.json();
  if (task.status === 'completed') return task.result;
  if (task.status === 'failed') throw new Error(task.error);
  return poll(taskId, delay);
}

const result = await poll(task_id);
console.log('Result:', result);
