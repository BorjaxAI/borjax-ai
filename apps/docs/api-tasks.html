API: Tasks

Create background AI tasks, poll for status, and retrieve results. Tasks run asynchronously via Celery and can take 15 seconds to 3 minutes depending on complexity.

Create Task

POST /v1/tasks Auth Required

Submit a new background task. Returns immediately with a task ID and pending status.

Request Body

json
{
  "type": "research",
  "prompt": "Research the competitive landscape of AI writing tools in 2026.",
  "agent_id": null
}
FieldTypeRequiredDescription
typestringYesOne of: research, write, analyze, scrape, custom
promptstringYesThe task instructions. Be specific for best results.
agent_idstring | nullNoRequired when type is custom. The agent's UUID.

Response 202 Accepted

json
{
  "id": "task_7f3a1b2c",
  "type": "research",
  "status": "pending",
  "prompt": "Research the competitive landscape...",
  "agent_id": null,
  "created_at": "2026-05-03T12:00:00Z"
}

Get Task

GET /v1/tasks/{id} Auth Required

Get the current status and result of a task. Poll this endpoint until status is completed or failed.

bash
curl https://api.borjaxai.com/v1/tasks/task_7f3a1b2c \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

Response 200 (completed)

json
{
  "id": "task_7f3a1b2c",
  "type": "research",
  "status": "completed",
  "prompt": "Research the competitive landscape of AI writing tools in 2026.",
  "result": "# AI Writing Tools Landscape 2026\n\n## Overview\n...",
  "error": null,
  "tokens_used": 4821,
  "agent_id": null,
  "created_at": "2026-05-03T12:00:00Z",
  "completed_at": "2026-05-03T12:01:47Z"
}

Response 200 (failed)

json
{
  "id": "task_7f3a1b2c",
  "status": "failed",
  "result": null,
  "error": "Bedrock service temporarily unavailable. Please retry.",
  "completed_at": "2026-05-03T12:00:45Z"
}

List Tasks

GET /v1/tasks Auth Required

Returns a paginated list of tasks for the authenticated user, ordered by most recent.

Query Parameters

ParamTypeDefaultDescription
statusstring—Filter by status: pending, running, completed, failed
typestring—Filter by task type
pageinteger1Page number
limitinteger20Results per page (max 50)
bash
# Get all completed research tasks
curl "https://api.borjaxai.com/v1/tasks?status=completed&type=research&page=1&limit=10" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

Status Lifecycle

text
  POST /v1/tasks
       ↓
  ┌──────────┐
  │ pending  │  Queued in Celery, waiting for worker
  └────┬─────┘
       ↓  worker picks up
  ┌──────────┐
  │ running  │  CrewAI crew executing (15s – 3min)
  └────┬─────┘
       ↓
  ┌──────────────────────┬───────────────────────┐
  │   completed          │   failed               │
  │   result: "..."      │   error: "..."         │
  │   tokens_used: N     │   tokens_used: partial │
  └──────────────────────┴───────────────────────┘

Full Polling Example (JavaScript)

Submit a task and poll until it completes, with exponential backoff:

javascript
const BASE = 'https://api.borjaxai.com/v1';
const headers = { 'Authorization': `Bearer ${ACCESS_TOKEN}`, 'Content-Type': 'application/json' };

async function submitTask(type, prompt, agentId = null) {
  const res = await fetch(`${BASE}/tasks`, {
    method: 'POST',
    headers,
    body: JSON.stringify({ type, prompt, agent_id: agentId }),
  });
  if (!res.ok) throw new Error(`Submit failed: ${res.status}`);
  return (await res.json()).id;
}

async function pollTask(taskId, intervalMs = 3000, maxAttempts = 60) {
  for (let i = 0; i < maxAttempts; i++) {
    await new Promise(r => setTimeout(r, intervalMs));

    const res = await fetch(`${BASE}/tasks/${taskId}`, { headers });
    const task = await res.json();

    console.log(`[${i + 1}] Status: ${task.status}`);

    if (task.status === 'completed') return task.result;
    if (task.status === 'failed') throw new Error(`Task failed: ${task.error}`);

    // Exponential backoff after 10 attempts
    if (i > 10) intervalMs = Math.min(intervalMs * 1.5, 15000);
  }
  throw new Error('Task timed out');
}

// Usage
const taskId = await submitTask('research', 'Research AI trends in healthcare for 2026.');
console.log('Task submitted:', taskId);
const result = await pollTask(taskId);
console.log('Result:', result);
